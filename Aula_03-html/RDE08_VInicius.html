RDE08_VInicius.html Tags

Table:

Uma tabela HTML simples, contendo duas colunas e duas linha!

Exemplo:

Month Savings
January $100

Del:

É um texto com uma parte excluída e uma nova parte inserida.

Exemplo:

My favorite color is blue red!

Ins:

É uma Tag que após ser feito a exclusão de uma parte do texto, com a tag "del", permite inserir uma parte nova.

Exemplo:

Mude seu vocabulário de "eu odeio" para "Eu amo"!

Sub:

É uma tag para textos subscritos.

Exemplo:

Esse texto possui textos subscritos.

Sup:

É uma tag para textos sobrescritos.

Exemplo:

Esse texto possui textossobrescritos.

Blockquote:

Uma seção que é citada de outra fonte.

Exemplo:

Clique aqui

Tudo está preparado aqui

casa e o meu coração também

Tu és o único motivo que me fez chegar

Os filhos já estão chegando aqui

Agora somos dois ou três, ou mais

Encontre o meu coração disposto a queimar por Ti

q:

Tag que possibilita marcar uma pequena citação.

Exemplo:

WWF's goal is to: Build a future where people live in harmony with nature. We hope they succeed.

Abbr:

É uma tag que marca abreviação.

Exemplo:

The WHO was founded in 1948.

Address:

Informações de contato para Example.com.

Exemplo:

Written by Jon Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

Cite:

Tag que define o título de um trabalho.

The Scream by Edward Munch. Painted in 1893.

Bdo:

Específica a direção do texto.

Exemplo:

This text will go right-to-left.

Code:

Define um texto como código de computador em um documento.

Exemplo:

The HTML button tag defines a clickable button.

The CSS background-color property defines the background color of an element.

Pre:

 Texto pré-formatado.

Exemplo:

            Text in a pre element
            is displayed in a fixed-width
            font, and it preserves
            both      spaces and
            line breaks

Kbd:

 texto como entrada de teclado em um documento.

Exemplo:

Press Ctrl + C to copy text (Windows).

Press Cmd + C to copy text (Mac OS).
